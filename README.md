# RetailOps 360

A retail analytics project analysing 100K+ real e-commerce orders
using SQL, Python, and Power BI.

## Dataset
Brazilian E-Commerce Public Dataset by Olist (Kaggle)
~100,000 orders | 2016–2018 | 9 relational tables

## Objectives
- Sales performance & demand trend analysis
- Market basket analysis (frequently bought together)
- Promotional ROI measurement
- KPI dashboard for business decision-making

## Tools
- MS SQL Server (data storage & querying)
- Python / Pandas / Jupyter (analysis & forecasting)
- Power BI (dashboard & visualization)
